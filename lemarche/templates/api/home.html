API marché de l'inclusion

L'API du marché de l'inclusion met à disposition le répertoire qualifié des structures d'insertion par l'activité économique.

Cet API est encore en construction, et est appelée à évoluer.

Version: 0.11

L'accès aux données complètes est limité pendant la période de construction du service.
Si vous désirez obtenir un jeton (token) utilisateur, merci de nous adresser votre demande sur lemarche@inclusion.beta.gouv.fr

Dernières Nouveautés

[v0.11] Préparation pour la v1
Les champs sont basculés en anglais pour homogénéiser avec le schéma sous-jacent. L'url est maintenant finale (sans le sous-domaine 'api').
[v0.10] Filtre secteur
Pouvoir filtrer les Siae par un ou plusieurs secteurs d'activité
[v0.9] Données "Type de structure" & "Type de prestation"
2 nouveaux endpoints "/siae/kinds" & "/siaes/presta-types". Possible aussi de filtrer les structures par type de prestation
Données "Réseau"
Un nouvel endpoint "/networks" donne la liste des réseaux de l'inclusion
Accès API déplacé
L'accès à l'API est déplacé dans le sous-dossiers /api
Secteurs d'activité
L'endpoint secteurs d'activité a été re-architecturé, les données sont désormais mieux représentées
